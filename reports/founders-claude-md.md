## Skill Description

A CLAUDE.md file is a persistent instruction set read by Claude Code at the start of every session. A founder's CLAUDE.md shapes Claude's operating behavior across their entire development workflow — task management, planning discipline, code style preferences, tool usage conventions, and communication norms.

Boris Cherny's public CLAUDE.md was used as the baseline for this evaluation. It is a well-structured, real-world example of how a senior practitioner instructs their AI collaborator.
